# 🪐 Solar System — Interactive Web Animation
 
An animated solar system that runs right in your browser. Every planet orbits the Sun at its own pace — no app to download, no account to create. Just open and watch.
 
## 🔴 Live Demo → [mr-rt1098.github.io/Solar-System](https://mr-rt1098.github.io/Solar-System/)
 
---
 
## How it's built
 
This project uses three core technologies that power most of the internet:
 
- **HTML5** — lays out everything you see on the page
- **CSS3** Animations — makes the planets move in smooth, continuous orbits
- **JavaScript** — brings interactivity and life to the whole thing
No frameworks. No shortcuts. Every line of code written by hand.
 
---
 
## The idea behind it
 
I gave myself a challenge — take something as complex as a moving solar system and build it using only the basics. Nothing pre-built, nothing borrowed.
 
What I got out of it was more than just a cool animation. I learned how to coordinate multiple elements moving at the same time, how browsers handle smooth motion, and how to keep code clean when things start getting complicated.
